# Arduino Circuits Collection

A comprehensive collection of Arduino circuits built and simulated in Tinkercad. This repository includes various IoT, sensor interfacing, display control, and motor control projects with complete documentation, source code, and circuit diagrams.

## Overview

This project showcases practical Arduino implementations covering:
- **Sensor Interfaces**: PIR, Ultrasonic, Temperature sensors
- **Display Controls**: LCD displays, 7-segment displays
- **Motor Control**: Servo motors, DC motors
- **Basic Components**: LEDs, pushbuttons, relays
- **Advanced Projects**: Multi-component integrated circuits

## Circuits Included

### Sensors & Detection
1. **PIR Sensor** - Motion detection circuit

4. **LCD Display Interfaces** - 16x2 LCD with various control methods

## Getting Started

### Prerequisites
- Arduino IDE (https://www.arduino.cc/en/software)
- Arduino Uno or compatible microcontroller
- Tinkercad account (for simulations): https://www.tinkercad.com
- Component libraries (included in Arduino IDE)

### How to Use

1. **Clone the repository**
   ```bash
   git clone https://github.com/MOHAMMADIKRAM03/Arduino-Circuits.git
   cd Arduino-Circuits
   ```

2. **Choose a circuit** from the folders

3. **Review the circuit documentation**
   - Check the README in each circuit folder
   - Review the circuit diagram
   - List of required components

4. **Upload the code**
   - Open the .ino file in Arduino IDE
   - Select your board (Tools > Board > Arduino Uno)
   - Select the correct COM port (Tools > Port)
   - Click Upload

5. **Build the physical circuit**
   - Follow the circuit diagram provided
   - Ensure all connections are correct
   - Double-check power connections

## Circuit Structure

Each circuit folder contains:
```
Circuit-Name/
├── README.md          # Circuit description and instructions
├── circuit.ino        # Arduino source code
├── circuit_diagram.png # Visual circuit diagram
└── components.txt     # Bill of materials
```
